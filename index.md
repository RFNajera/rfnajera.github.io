# Hello
{:.no_toc}
My name is René F. Najera, DrPH, and I am an epidemiologist. What is an epidemiologist? We're the people who study things that happen to populations. We take data from many different sources, analyze them, and come up with the conclusions necessary to determine strategies to improve public health.

When I was working at a state health department, I did most of my data analysis with [Microsoft Excel and Microsoft Access](https://products.office.com/en-us/home). This is fine and good, but those software packages are limited. There was much more I wish I could do. Once I started working on my doctoral degree (DrPH), I was introduced to software like [STATA](https://www.stata.com/) for data analysis and biostatistical analysis. I was also introduced to [ArcGIS](https://www.arcgis.com/index.html), a software package for geographical and geostatistical analysis.

I was only recently introduced to the [R Programming Language](https://www.r-project.org/about.html). R is an open-source programming language that allows the user to perform a lot of data management, statistical analysis, and even geographic/geostatistical analyses. As I've been learning R, it's open-source nature and generally free cost has convinced me that this is a good software package to teach epidemiologists the world over.

With that in mind, I decided to improve my skills in R programmig by creating some projects that public health professionals can use. I'm using open-source data as well, allowing anyone with an internet connection to follow along and see how this is done. If you have any questions, please feel free to contact me on [Twitter (@Epiren)](http://twitter.com/epiren) or [Facebook](http://facebook.com/rene.f.najera).

# Projects

## Analysis of Inequities and Homicides in Baltimore

This project is a quick-ish overview of societal inequities in Baltimore and their association with homicides. Baltimore has historically been one of the most violent cities in the United States ([and the world](http://www.seguridadjusticiaypaz.org.mx/seguridad/1567-estudio-las-50-ciudades-mas-violentas-del-mundo-2018)).

You can access the project [by clicking here](https://rfnajera.github.io/baltimore-homicide-inequity/).
